# AttackArea
一个基于 OpenGL 和 Creator 的攻击范围绘制Demo。
使用 GLSL 和 JavaScript 编写

出于贴图的不灵活性，编写了此 shader
用于节省贴图资源以及提高灵活性

现支持圆环和扇形的攻击范围绘制
支持编辑器配置颜色，攻击范围大小，扇形朝向，饱和度等

附带一个简易的虚拟摇杆，资源来自网络
